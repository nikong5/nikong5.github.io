基本信息
初始 B 名称 露西亚·红莲 武器 太刀
类型 进攻 属性 物理(主要)/火 距离 近
生命 1113 核心 任意三消+红 攻击 223
防御 159 定位 主C 会心 101
信号球
红莲-起手式
朝前突刺，拉扯命中的目标，造成200%/400%/600%的物理伤害。
红莲-闪击
向目标快速发动1次攻击，击飞被攻击的目标，并造成150%/300%/450%的物理伤害。
红莲-波动斩
武器爆发，根据消球数使武器持续4/4/6秒的时间内所有攻击附加10%/15%/20%的火属性伤害。
主动技能
红莲-迅刀式
普通攻击：使用红莲迅刀式连续攻击敌人，共造成540%的物理伤害。
必杀-红莲乱舞
必杀技：消耗100点能量，释放红莲剑舞快速对前方敌人进行斩击，每击造成140%的的物理伤害，最后对敌人进行重击造成1000%的物理伤害。
莲华快斩
QTE：露西亚出场使用双刀术攻击敌人，共造成共造成740%的物理伤害。
被动技能
进攻型
伤害增加20%。
红莲-双刀术
触发条件：任意三消 +
核心被动：任意3消后，4秒内再次消除红色信号时，露西亚进入爆发状态，将使用双刀术代替普通攻击。红莲-双刀术：连续攻击敌人，共造成920%的物理伤害。
队长-无畏战士
队长技：全队进攻型构造体攻击增加10%。
闪击-气刃斩构造体品质到达SS解锁
被动：攻击击飞的目标时，每次攻击额外附加20%的物理伤害
不止之刀构造体品质到达SSS解锁
被动：双刀爆发状态下，全伤害增加15%。
明镜核心构造体品质到达SSS+解锁
被动：普通攻击伤害增加15%。
终阶解放
首次进入战斗时，立即获得3个信号球。
专属武器

天狼星
武器效果: 增加5%全伤害，消球时会根据消球数量累积能量，50点能量后，召出天狼星，存在45秒，冷却时间60秒。
